PRINCIPALES AMENAZAS

  • Phishing: Técnicas de engaño para obtener información confidencial a través de correos electrónicos o sitios web falsos.
  • Malware: Software malicioso diseñado para dañar o comprometer sistemas, incluyendo virus, gusanos y troyanos.
  • Ransomware: Malware que cifra archivos y exige un rescate para su liberación.
  • Attacks de Denegación de Servicio (DDoS): Ataques que sobrecargan un servidor o red para hacerlo inaccesible.
  • Exploits: Uso de vulnerabilidades de software para ejecutar código malicioso en un sistema.
  • Inyección SQL: Técnica que permite a los atacantes ejecutar comandos maliciosos en bases de datos a través de formularios web.
  • Robo de Identidad: Suplantación de la identidad de una persona para acceder a información sensible.
  • Spyware: Software que recoge información sin el consentimiento del usuario, como datos de navegación o contraseñas.
  • Redes de Botnet: Redes de dispositivos comprometidos controlados por un atacante para realizar tareas maliciosas.
