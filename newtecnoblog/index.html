CONCEPTOS...

Pagina Web

Es un documento digital accesible a través de un navegador web mediante una URL. Puede contener texto, imágenes, videos, enlaces y otros elementos multimedia. Generalmente, forma parte de un sitio web más grande.

Sitio Web

Es un conjunto de páginas web organizadas bajo un mismo dominio, conectadas entre sí mediante enlaces. Puede tener distintos propósitos, como informar, vender productos, ofrecer servicios o permitir interacción con los usuarios. Ejemplos incluyen blogs, tiendas en línea y redes sociales.

Aplicacion Web

Es un tipo de sitio web interactivo diseñado para realizar funciones específicas, similar a una aplicación de escritorio o móvil. Se ejecuta en un navegador y permite a los usuarios ingresar datos, procesar información y obtener resultados en tiempo real. Ejemplos incluyen Gmail, Google Docs y redes sociales como Facebook.
